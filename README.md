# 🎬 Movie List App

A modern web app to search and browse your favorite movies. Built with Vite, JavaScript, and a clean dark UI.

## 🚀 Features

- Browse trending and popular movies
- Clean modern UI with dark mode
- Responsive design

## 🛠 Tech Stack

- Vite/React
- JavaScript
- HTML/CSS

## 📦 Setup

```bash
npm install
npm run dev
